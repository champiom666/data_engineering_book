# Part 6: Reasoning and Agent Data Engineering

## Positioning

Part 6 focuses on reasoning traces, tool use, function calling, agent memory, and multi-turn interaction samples. It discusses how complex reasoning and action processes can be converted into trainable, verifiable, and reviewable data assets.

- [Chapter 18: Chain-of-Thought and Reasoning Data Engineering](ch18_cot.md)
- [Chapter 19: Tool-Use and Function Calling Data](ch19_tool.md)

## Suggested Reading Order

- Read Chapter 18 first to understand CoT and reasoning-trace sample design.
- Then read Chapter 19 to master tool calls, function signatures, and execution constraints.
- Finally read Chapter 20 to extend the discussion to agent memory, multi-turn state, and long-term interaction data.
